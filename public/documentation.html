Movie API Documentation

Welcome to the documentation for the Movie API. This API provides access to information about movies, actors, directors, and genres.

API Endpoints

Endpoint Method Description
/movies/title/:title GET Returns data (director, genre, actors) about a single movie by its title.
/movies/genre/:genreName GET Returns a list of movies by genre.
/movies/director/:directorName GET Returns a list of movies by director.
/movies/:movieId GET Returns data (director, genre, actors) about a single movie by its ID.
/movies POST Creates a new movie. Requires a JSON request body with movie details.
/movies/:movieId PUT Updates movie info by ID. Requires a JSON request body with updated movie details.
/movies/:movieId DELETE Deletes a movie by ID.
/movies/:movieId/cast GET Returns the cast (actors) of a movie.
/actors/movie/:movieId GET Returns a list of actors for a specific movie.
/actors/genre/:genreName GET Returns a list of actors who have appeared in movies of a specific genre.
/actors/name/:name GET Returns data about a single actor by their name.
/actors/:actorId GET Returns data about a single actor by their ID.
/actors GET Returns a list of all actors.
/actors POST Creates a new actor. Requires a JSON request body with actor details.
/actors/:actorId PUT Updates actor info by ID. Requires a JSON request body with updated actor details.
/actors/:actorId DELETE Deletes an actor by ID.
/genres GET Returns a list of all genres.
/genres/:name DELETE Deletes a genre by name.
/genres/:name PUT Updates a genre's description by name.
/directors GET Returns a list of all directors.
/directors/name/:name GET Returns data about a single director by their name.
/directors/:directorId GET Returns data about a single director by their ID.
/users POST Registers a new user. Requires a JSON request body with user details.
/users/:username PUT Updates user information. Requires a JSON request body with updated user details.
/users/:username/favorites/:movieId POST Adds a movie to a user's list of favorite movies.
/users/:username/favorites/:movieId DELETE Removes a movie from a user's list of favorite movies.
/users/:username DELETE Deregisters a user.
/ GET Welcome message.
/admin/users GET (Admin) Returns a list of all users.
/admin/movies GET (Admin) Returns a list of all movies with details.
/admin/genres GET (Admin) Returns a list of all genres.
/admin/directors GET (Admin) Returns a list of all directors.
/admin/actors GET (Admin) Returns a list of all actors.

Endpoint Details

Get movie by title

GET /movies/title/:title

Returns a single movie object that matches the title provided in the URL.

Example: /movies/title/Die%20Hard

Response: 

                    {
                      "movieId": 1,
                      "title": "Die Hard",
                      "description": "A New York cop battles terrorists in a Los Angeles skyscraper.",
                      "director": {
                        "directorid": 200001,
                        "name": "Christopher Nolan",
                        "bio": "British-American film director known for his complex, mind-bending plots and visual style.",
                        "birthyear": "1970-07-30T00:00:00.000Z",
                        "deathyear": null
                      },
                      "genre": {
                        "genreid": 100001,
                        "name": "Action",
                        "description": "Films involving high-stakes, physical feats and battles."
                      },
                      "imageURL": "diehard.png",
                      "featured": true,
                      "releaseYear": 1988,
                      "rating": 8.2,
                      "actors": [
                        {
                          "actorid": 4000003,
                          "name": "Bruce Willis",
                          "bio": "American actor known for his role in Die Hard",
                          "birthdate": "1955-03-19T00:00:00.000Z",
                          "deathdate": null,
                          "pictureurl": "bruce_willis.jpg"
                        }
                      ]
                    }

Status Codes:

  • 200 OK - Success
  • 404 Not Found - Movie not found
  • 500 Internal Server Error - Server error

Create new movie

POST /movies

Creates a new movie in the database.

Request Body Example: 

                    {
                      "title": "New Movie Title",
                      "description": "Description of the new movie.",
                      "directorid": 200001,  // Director ID from the directors table
                      "genreid": 100001,     // Genre ID from the genres table
                      "imageurl": "new_movie.jpg",
                      "featured": true,
                      "releaseyear": 2024,
                      "rating": 7.5
                    }

Response: 

                    {
                      "movieid": 21,
                      "title": "New Movie Title",
                      "description": "Description of the new movie.",
                      "directorid": 200001,
                      "genreid": 100001,
                      "imageurl": "new_movie.jpg",
                      "featured": true,
                      "releaseyear": 2024,
                      "rating": 7.5
                    }

Status Codes:

  • 201 Created - Movie created successfully
  • 500 Internal Server Error - Server error

Register new user

POST /users

Registers a new user in the application.

Request Body Example: 

                    {
                      "username": "newuser",
                      "email": "newuser@example.com",
                      "firstname": "New",
                      "lastname": "User"
                    }

Response: 

                    {
                      "userid": 300006,
                      "username": "newuser",
                      "email": "newuser@example.com",
                      "firstname": "New",
                      "lastname": "User"
                    }

Status Codes:

  • 201 Created - User registered successfully
  • 400 Bad Request - Username already exists
  • 500 Internal Server Error - Server error

Update user info

PUT /users/:username

Updates the information for an existing user.

Request Body Example: 

                    {
                      "email": "updateduser@example.com",
                      "firstname": "Updated",
                      "lastname": "User"
                    }

Response: 

                    {
                      "userid": 300006,
                      "username": "newuser",
                      "email": "updateduser@example.com",
                      "firstname": "Updated",
                      "lastname": "User"
                    }

Status Codes:

  • 200 OK - User information updated successfully
  • 404 Not Found - User not found
  • 500 Internal Server Error - Server error

Add movie to favorites

POST /users/:username/favorites/:movieId

Adds a movie to a user's list of favorite movies.

Example: /users/existinguser/favorites/123

Response: 

                    Movie added to favorites.

Status Codes:

  • 200 OK - Movie added to favorites
  • 400 Bad Request - Movie already in favorites
  • 404 Not Found - User or movie not found
  • 500 Internal Server Error - Server error

Data Objects

Movie Object

Represents a movie in the API. Note that director and genre are represented by their IDs, and actors are related through the moviecast table. 

                      {
                        "movieId": 1,         // Integer (Primary Key)
                        "title": "Die Hard",  // String
                        "description": "A New York cop battles terrorists in a Los Angeles skyscraper.", // String
                        "directorid": 200001,  // Integer (Foreign Key to directors table)
                        "genreid": 100001,     // Integer (Foreign Key to genres table)
                        "imageURL": "diehard.png", // String (URL)
                        "featured": true,      // Boolean
                        "releaseYear": 1988,   // Integer
                        "rating": 8.2         // Double
                      }

Genre Object

Represents a genre of movies.

                      {
                        "genreid": 100001,   // Integer (Primary Key)
                        "name": "Action",    // String
                        "description": "Films involving high-stakes, physical feats and battles." // String
                      }

Director Object

Represents a director of movies.

                      {
                        "directorid": 200001,  // Integer (Primary Key)
                        "name": "Christopher Nolan", // String
                        "bio": "British-American film director...", // String
                        "birthyear": "1970-07-30T00:00:00.000Z", // Date (or String in some responses)
                        "deathyear": null       // Date (or String/null)
                      }

Actor Object

Represents an actor in movies.

                      {
                        "actorid": 4000003,  // Integer (Primary Key)
                        "name": "Bruce Willis", // String
                        "bio": "American actor...", // String
                        "birthdate": "1955-03-19T00:00:00.000Z", // Date (or String)
                        "deathdate": null,     // Date (or String/null)
                        "pictureurl": "bruce_willis.jpg" // String (URL)
                      }

User Object

Represents a user of the application.

                      {
                        "userid": 300006,  // Integer (Primary Key)
                        "username": "newuser", // String
                        "email": "newuser@example.com", // String
                        "firstname": "New",    // String
                        "lastname": "User"     // String
                      }
