Movie API Documentation

Welcome to the documentation for the Movie API. This API provides access to information about movies, actors, directors, and genres.

API Endpoints

Endpoint Method Description
/movies GET Retrieves a list of all movies.
/movies/title/:title GET Finds a movie by its title (case-insensitive search).
/movies/genre/:genreName GET Retrieves movies by genre name.
/movies/director/:directorName GET Retrieves movies by director name.
/movies/:movieId GET Finds a movie by its unique ID.
/movies POST Creates a new movie.
/movies/:movieId PUT Updates a movie by its ID.
/movies/:movieId DELETE Deletes a movie by its ID.
/movies/:movieId/actors GET Retrieves the list of actors in a movie.
/users POST Registers a new user.
/users/:username PUT Updates user information.
/users/:username/favorites/:movieId POST Adds a movie to user's favorites.
/users/:username/favorites/:movieId DELETE Removes a movie from user's favorites.
/users/:username DELETE Deletes a user.
/genres GET Retrieves a list of all genres.
/genres/:name GET Finds a genre by its name.
/genres POST Creates a new genre.
/genres/:name PUT Updates a genre's description.
/genres/:name DELETE Deletes a genre.
/directors GET Retrieves a list of all directors.
/directors/name/:name GET Finds a director by their name.
/directors/:directorId GET Finds a director by their ID.
/directors POST Creates a new director.
/directors/:directorId PUT Updates a director's information.
/directors/:directorId DELETE Deletes a director.
/actors GET Retrieves a list of all actors.
/actors/name/:name GET Finds an actor by their name.
/actors/:actorId GET Finds an actor by their ID.
/actors POST Creates a new actor.
/actors/:actorId PUT Updates an actor's information.
/actors/:actorId DELETE Deletes an actor.
/admin/users GET (Admin) Retrieves a list of all users.
/admin/movies GET (Admin) Retrieves a list of all movies with details.
/admin/genres GET (Admin) Retrieves a list of all genres.
/admin/directors GET (Admin) Retrieves a list of all directors.
/admin/actors GET (Admin) Retrieves a list of all actors.
/ GET Provides a welcome message.

Data Objects

Movie Object

                            {
                                "_id": "string (ObjectId - a unique identifier for each movie)",
                                "title": "string (required - the movie's title)",
                                "description": "string (required - a brief summary of the movie)",
                                "genre": {
                                    "name": "string (the movie's genre, e.g., 'Comedy', 'Drama')",
                                    "description": "string (a description of the genre)"
                                },
                                "director": {
                                    "name": "string (the director's name)",
                                    "bio": "string (the director's biography)",
                                    "birth": "date (ISO 8601 format, e.g., '1990-01-01')",
                                    "death": "date (ISO 8601 format, optional - if the director is deceased)"
                                },
                                "actors": [
                                    {
                                        "_id": "string (ObjectId - a unique identifier for each actor)",
                                        "name": "string (actor name)"
                                    }
                                ],
                                "imagePath": "string (optional - a link to the movie's poster image)",
                                "featured": "boolean (optional - whether the movie is featured or not)",
                                "__v": "number (version key - used internally by the database)"
                            }

User Object

                            {
                                "_id": "string (ObjectId - a unique identifier for each user)",
                                "username": "string (required - the user's login name)",
                                "email": "string (required - the user's email address)",
                                "birthday": "date (ISO 8601 format, optional - user's birthday, e.g., '1985-05-20')",
                                "favoriteMovies": [
                                    {
                                        "_id": "string (ObjectId - a unique identifier for each movie)",
                                        "title": "string (movie title)"
                                    }
                                ],
                                "firstname": "string (optional - user's first name)",
                                "lastname": "string (optional - user's last name)",
                                "__v": "number (version key - used internally by the database)"
                            }

Genre Object

                            {
                                "_id": "string (ObjectId - a unique identifier for each genre)",
                                "name": "string (required - the genre's name, e.g., 'Action')",
                                "description": "string (required - a description of the genre)",
                                "__v": "number (version key - used internally by the database)"
                            }

Director Object

                            {
                                "_id": "string (ObjectId - a unique identifier for each director)",
                                "name": "string (required - the director's name)",
                                "bio": "string (required - the director's biography)",
                                "birth": "date (ISO 8601 format, required - director's birthdate, e.g., '1970-12-01')",
                                "death": "date (ISO 8601 format, optional - director's date of death, e.g., '2023-06-15', if applicable)",
                                "__v": "number (version key - used internally by the database)"
                            }

Actor Object

                            {
                                "_id": "string (ObjectId - a unique identifier for each actor)",
                                "name": "string (required - the actor's name)",
                                "bio": "string (required - the actor's biography)",
                                "birth": "date (ISO 8601 format, required - actor's birthdate, e.g., '1965-08-10')",
                                "death": "date (ISO 8601 format, optional - actor's date of death, e.g., '2024-03-22', if applicable)",
                                "pictureUrl": "string (optional - a link to the actor's picture)",
                                "__v": "number (version key - used internally by the database)"
                            }
